### Bugs Fixed During Deployment (commits ecb7a52 → 771ba14)
1. **Upstash JSON auto-parse** — `redis.get()` returns parsed objects, not strings. Added `typeof` check before `JSON.parse()`
2. **Lua `tonumber()` overflow** — `tonumber('10000000000000000000')` returns nil in Lua (exceeds float64 precision). Replaced wei-based daily limits with simple bootstrap counter (`DAILY_BOOTSTRAP_LIMIT=500`)
3. **Lua nil handling** — Redis `GET` on missing key returns `false` not nil in Lua. Added explicit `if raw and raw ~= false then` guard
4. **Treasury key format** — `privateKeyToAccount()` requires `0x` prefix. Env var was stored without it (or with trailing whitespace). Added `.trim()` + `0x` prefix normalization
5. **Serverless state** — Lazy-init wallet clients cached in module scope don't persist across Vercel invocations. Changed to create clients per-request via `getTransferClients()`
6. **Error reporting** — Added `detail` field to 500 responses for debugging in non-production
